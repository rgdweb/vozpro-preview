# VozPro - Script de Diagnóstico + Auto-Restart

## O que faz

### Diagnóstico completo (10 verificações):
1. **GPU NVIDIA** - VRAM, temperatura, uso, versão driver
2. **Servidor Gradio** - Porta 7861, PID, tempo de resposta
3. **Cloudflared Tunnel** - Processo ativo, acessível via internet
4. **Disco** - Espaço livre nos drives C: e D:
5. **Memória RAM** - Total, disponível, percentual usado
6. **Internet** - Conectividade e latência
7. **Python + Dependências** - PyTorch, OmniVoice, Gradio, CUDA
8. **Arquivos Temporários** - Contagem e limpeza automática
9. **Áudios de Referência** - Integridade e tamanho
10. **Riscos Futuros** - Problemas potenciais detectados

### Auto-Restart:
- Monitora a fila de geração via API Vercel
- Quando não há ninguém na fila por X minutos → restart completo
- Limpa temp → para Gradio → para Tunnel → reinicia tudo
- Máximo 5 restarts por dia (proteção)
- Log de tudo em `diagnostico.log`

## Como usar

### 1. Só diagnóstico (uma vez)
```
python diagnostico_auto_restart.py --diag
```

### 2. Monitoramento contínuo (com auto-restart)
```
python diagnostico_auto_restart.py --monitor
```
Ou simplesmente:
```
python diagnostico_auto_restart.py
```
(responde S quando perguntar)

### 3. Forçar restart agora
```
python diagnostico_auto_restart.py --restart
```

## Configuração

Edite o dicionário `CONFIG` no topo do script:

| Config | Padrão | Descrição |
|--------|--------|-----------|
| `gradio_port` | 7861 | Porta do Gradio |
| `auto_restart_enabled` | True | Ativar auto-restart |
| `idle_minutes_before_restart` | 60 | Minutos ocioso antes de restart |
| `check_interval_seconds` | 120 | Intervalo entre verificações |
| `max_restarts_per_day` | 5 | Limite de restarts por dia |
| `cleanup_enabled` | True | Limpar arquivos temporários |
| `temp_max_age_hours` | 2 | Idade máxima dos arquivos temp |
| `gpu_temp_warning` | 80°C | Alerta de temperatura GPU |
| `gpu_vram_warning` | 90% | Alerta de uso VRAM |

## Configuração de Email (para envio de áudio)

Para que a opção "Receber via E-mail" funcione, configure as variáveis de ambiente no Vercel:

### Usando Gmail (grátis):
1. Vá em https://myaccount.google.com/apppasswords
2. Crie uma senha de app
3. No painel do Vercel (Settings > Environment Variables), adicione:

```
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=seu-email@gmail.com
SMTP_PASS=sua-senha-de-app
EMAIL_FROM=VozPro <seu-email@gmail.com>
```

### Usando Outlook/Hotmail:
```
SMTP_HOST=smtp-mail.outlook.com
SMTP_PORT=587
SMTP_USER=seu-email@outlook.com
SMTP_PASS=sua-senha
EMAIL_FROM=VozPro <seu-email@outlook.com>
```

### Testando:
- GET `/api/send-audio-email` → verifica se está configurado
- Se `configured: false`, o botão de email aparece desativado
